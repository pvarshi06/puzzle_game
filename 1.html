Instructions for the game

1.The game consists of a question and if you give correct answer you will win the game.

2.For each letter of the answer you will be given hints.If your letter is correct with first hint then it will redirect to next question which will be your next letter.

3.If letter for clue is wrong with second attempt then you will lost the game and if it is correct with second hint it redirects to next question.

4.The final Answer should be in CAPITAL LETTERS.

5.The answer will be according to order of the clues.
